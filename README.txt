In order for the mod to work you need:
1) Move folder ukraine-main to : C:\Users\User\Documents\Paradox Interactive\Hearts of Iron IV\mod (Usually this path, but not always
2) Separately copy file descriptor.mod from folder ukraine-main to folder Hearts of Iron IV\mod and rename it to ukraine-main.mod
3) Open the file ukraine-main.mod in any text editor and at the very end add the line path="(your path to the folder mod)"
4) Next, open the Hoi4 Launcher and under "Game Sets" and click on "add other modifications" and look for ukraine mod there and add it to the set
5) Play
