Скрипты используют ffmpeg для работы с аудиофайлами.

Что нужно сделать перед использованием:
1. Распаковать файлы из архива с ffmpeg в любую папку на жёстком диске (НО НЕ В ПАПКУ МОДА)
2. В меню "Пуск" найти "Изменение системных переменных среды"
3. На вкладке "Дополнительно" в правом нижнем углу нажать на "Переменные среды..."
4. В системных переменных (снизу) найти переменную "Path". Нажать на неё (выбрать), а затем "Изменить..."
5. Справа сверху нажать кнопку "Создать"
6. В появившееся поля ввода внизу списка вписать путь до bin папки, в которой лежат три файла: ffprobe.exe, ffplay.exe, ffmpeg.exe. (Эта папка находилась внутри архива, который вы ранее распаковали).

Пример того, как должен выглядеть итоговый путь в Path, приведён в файле "example_ffmpeg_path.jpg", лежащем рядом с README.TXT

=====================================
Использование скриптов:
=====================================

reencode_ogg_192.py

Рекурсивно ищет *.ogg в соседней папке 'music' и перекодирует аудио в 192 kbps,
если текущий битрейт выше 192 kbps.

Использование:
  python reencode_ogg_192.py
  python reencode_ogg_192.py --dry-run        # только показать, что будет сделано
  python reencode_ogg_192.py --verbose        # подробный вывод
  
=======
